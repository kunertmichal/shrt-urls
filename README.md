## Description

This app is a backend for an app that allows users to shorten URLs. It is built with NestJS and uses Redis to store the shortened URLs.

Next.js client is [here](https://github.com/kunertmichal/shrt-url-next)

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
